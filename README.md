# Search Engine
## By Mugagga Kimera

### Install
### Unix / Macos
#### Run these commands
```bash
python3 -m venv .venv
python -m pip install -r requirements.txt 
source .venv/bin/activate
```

### Windows
#### Run these commands
```bash
py -m venv .venv
python -m pip install -r requirements.txt 
.venv/bin/activate
```

### Usage
#### By 
#### Unix / Macos
```bash
python main.py
```

#### Windows
```bash
py main.py
```
